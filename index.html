Projetos

CRUD usando NodeJs

https://github.com/AndreXime/EstoqueExpress

Desenvolvi um sistema web completo chamado EstoqueExpress, voltado para a gestão de estoque, utilizando as tecnologias mais modernas de desenvolvimento fullstack. O projeto conta com as seguintes funcionalidades:

  • Backend: Implementação do servidor com Express.js para gerenciar as rotas e comunicações com o banco de dados. Utilizei o MongoDB para armazenar e gerenciar os dados de produtos, usuários e estoque.
  • Frontend: Usando EJS (Embedded JavaScript), as páginas dinâmicas são renderizadas diretamente no servidor, integrando o serviços de APIs para uma experiência contínua do usuário. Para o design, utilizei Bootstrap, garantindo uma interface moderna e responsiva.
  • Autenticação e Segurança: Integrei autenticação de usuários com sessões, garantindo a proteção dos dados através de controle de acesso, além de medidas para proteção contra acessos não autorizados.
