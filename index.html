CRUD usando NodeJs

https://github.com/AndreXime/UserManagementDjango

Desenvolvi um website CRUD (Create, Read, Update, Delete) utilizando Express.js para o backend e Bootstrap para o frontend, com o objetivo de criar uma aplicação web. Durante o desenvolvimento deste projeto, usei habilidades em gerenciamento de status HTTP. Além disso, aprendi a utilizar o método fetch do JavaScript para realizar requisições HTTP assíncronas, o que melhorou a comunicação entre o frontend e o backend da aplicação.

  • Frameworks e Tecnologias: Utilizei Express.js para construir o servidor e gerenciar as rotas, e Bootstrap para estilizar e criar uma interface de usuário responsiva e atraente.
  • Fetch API: Implementei o método fetch do JavaScript para realizar chamadas de API assíncronas, o que permitiu a atualização dinâmica de dados na página sem a necessidade de recarregar o navegador.
